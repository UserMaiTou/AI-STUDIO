# PROJECT_ARTIFACTS

## Purpose

AI maintains this file during project execution.

Use it to record assets created during the project so future reviews can identify which assets repeatedly appear across projects.

## What AI Must Record

- new directories
- new documents
- new checklists
- new templates
- new research materials
- new workflow documents
- new configuration documents

## Artifact Entries

```text
Date:
Artifact:
Type: directory | document | checklist | template | research | workflow | configuration | other
Path:
Created by:
Purpose:
Reused from:
Future reuse potential:
Notes:
```
