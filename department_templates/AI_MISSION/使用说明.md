# 部门 AI 使用快速指南（<DepartmentName>）

> 这是面向部门人员的通俗使用说明，是 `AI_MISSION/` 包的入口文档之一，与
> `START_HERE.md` 配套：`START_HERE.md` 给 AI 读流程，本文件给「人」看怎么用。
>
> 这是 canonical 模板（department-neutral）。导出时替换下面占位符，并删除本引用块：
> - `<DepartmentName>` — 部门名（如 `研发部`、`换皮部`）。
> - `<ProjectKind>` — 工作类型词（如 `研发`、`换皮`）。
> - `<MissionPack_v1>` — 本包的 Mission Pack 文件夹名（如 `GameRND_v1`、`Reskin_v1`）。

## 一句话：这是什么

这是一个让 AI 帮你做项目的「工具包」。你不需要懂 AI，只要会复制粘贴、能说清楚你要做什么，就能用。

## 第一步：拿到包之后怎么开始

1. 把整个 `AI_MISSION/` 文件夹复制到你的项目文件夹里。
2. 打开 **Claude**。
3. 把下面【开场白模板】整段复制给 Claude。
4. 之后跟着 Claude 一步步做就行——它会带你走。

## 四个 AI 各管什么（简单版）

- **ChatGPT**：帮你和老板把「要做什么」想清楚（出主意、定方向）。
- **Claude**：真正干活的主力。把想清楚的任务交给它，它来执行并自动记录。
- **Codex**：第二双眼睛，帮忙复查（顾问，不是主力）。
- **Cursor**：需要时帮忙批量改文件 / 写代码。

一句话记住：**想清楚找 ChatGPT 或老板，干活找 Claude。**

## 推荐流程（最省事）

1. 先用 ChatGPT，或直接和老板一起，把任务想清楚，凑成一个「任务包」（要做什么、涉及哪些东西、做到什么算完成）。
2. 打开 Claude，把任务包**一次性**交给它。
3. Claude 会自己判断任务大小，能做完的一口气做完，该停下问人的地方会停下来问你。

## Claude 会自己判断任务大小（你不用记）

Claude 内置一个「轻重判断」，你只要知道结果：

- 很小的问题（问个事、翻译、解释）→ 直接回答。
- 小改动（改 1–3 个文件）→ 直接做完给你结果。
- 中等任务 → 读必要资料、做一次检查后完成。
- 高风险（导出、发布、删改重要东西、涉及钱 / 账号）→ 更谨慎，并在关键处停下来问人。

## 哪些能一口气做，哪些必须停下来问人

**能一口气做：**
- 小修改、补文档、整理记录、回答问题。

**必须停下来问人：**
- 要花很多时间，或范围明显变大。
- 涉及钱、账号、签名、隐私、广告、上架、发布。
- 要删除或大改重要文件。
- 要动项目里已有的 `AI_STUDIO/`。
- 需要老板拍板的方向选择。

记住：**该停的时候 Claude 会自己停，你只管回答它的问题。**

## 直接复制：Claude 新会话开场白模板

把下面整段复制给 Claude：

```text
我们要用 AI_MISSION 开始一个新的 <ProjectKind> 任务。
请先读 AI_MISSION/START_HERE.md、AI_MISSION/AGENTS.md、AI_MISSION/core/AI_ROLE_ROUTING.md、AI_MISSION/<MissionPack_v1>/MISSION.md。
如果这个项目已经有 AI_STUDIO/，就以项目里的 AI_STUDIO/ 为准。
先停在 Phase 0 只读检查，等我明确同意再动手改东西。
第一步：帮我把 AI_MISSION/PROJECT_CONTEXT.md 填好。
然后开始第一轮 <ProjectKind> 规划。
```

## 任务包模板（想清楚再交给 Claude）

复制下面，填好再发给 Claude：

```text
任务目标：（一句话说清要做成什么）
背景 / 为什么做：
涉及文件或范围：（知道就写，不知道可留空）
做到什么算完成：（验收标准）
不要碰的东西：（限制）
有没有时间 / 上线要求：
```

## 做完之后，怎么反馈给老板（Human Director）

请把这几样回给老板：

1. **结果**：做完了什么。
2. **问题**：哪里卡住、哪里没做完。
3. **回传**：把整个 `AI_MISSION/` 文件夹交回（里面有 AI 自动记录的全过程）。
4. **使用感受**：好不好用、顺不顺手。
5. **读不懂的地方**：哪段说明看不明白。
6. **token 是否异常**：有没有觉得 AI 跑得特别久 / 特别费？有的话截图或简单说明。

## 一句话总结

想清楚 → 复制开场白给 Claude → Claude 一口气做或停下问你 → 做完把 `AI_MISSION/` 交回老板，并说说哪里不好用。
