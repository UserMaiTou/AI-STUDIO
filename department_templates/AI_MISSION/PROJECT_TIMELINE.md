# PROJECT_TIMELINE

## Purpose

AI maintains this file during project execution.

Use it to record time cost, blockers, waiting periods, and milestones so future Mission Pack reviews can identify slow or fragile workflow steps.

## What AI Must Record

- phase start time
- phase end time
- blocker duration
- waiting time
- important milestones

## Timeline Entries

```text
Date:
Workflow phase:
Event type: phase-start | phase-end | blocker | waiting | milestone | other
Start time:
End time:
Duration:
Reason:
Impact:
Next action:
```

