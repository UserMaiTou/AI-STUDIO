# GAME_RND_ESCALATION_RULES

> Purpose: define topics that require Human Director Review during new game R&D.

No Agent may make final decisions on the topics below.

---

## Must Escalate To Human Director

Escalate before proceeding when any of the following occur:

- new project direction change
- core gameplay reversal
- business model change
- need for large new system expansion
- major technical direction change
- need for formal project approval
- need to pause the project
- need to cancel the project
- account decisions
- SDK decisions
- privacy decisions
- compliance decisions
- key monetization decisions
- commit, push, release, or publishing authorization

---

## Escalation Format

```text
Issue:
Current evidence:
Objective validation already checked:
Options:
Recommended option:
Risk if delayed:
Decision needed from Human Director:
```

---

## Principle

Agents may collect information, inspect files, run approved validation, prepare options, draft checklists, and identify risks.

AI should complete practical objective validation before escalating.

Human Director remains responsible for subjective judgment, strategic direction, protected decisions, final authorization, and risk acceptance.

---

## If Unsure

Stop and mark:

```text
NEEDS_HUMAN_REVIEW
```

Do not proceed by assumption.
